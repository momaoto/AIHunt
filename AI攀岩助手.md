# 功能

核心：线路识别与示范

1. AI示范（技术功能）
- 月亮板，K板：输入线路序号，可视化线路结果
- 其他岩馆自己定的线：拍照上传ai识别+手动标记岩点，可视化线路结果
- 先用K板这样的公开线路测试，确保成功之后，确认无版权问题后导入岩馆自己定的线路
2. 真人示范（社区功能）
- 其他用户爬过之后可以做线路分享，视频示范，图文经验


其他：

3. 记录每一条线爬了多少次，视频复盘
4. 根据1，推算能量消耗，提供饮食建议
5. 接入岩馆端，用AI辅助定线，尝试规范化线路难度（各个岩馆难度标准差距很大，思路是与标准K板做某种量化对比，计算出更统一的难度）
6. 次卡分享（类似：微信小程序cika）
7. isaacsim式的仿真，做成3d的可视化形式
8. 待续

# 技术
图像识别与分割（识别岩点），人体动作生成（用于生成攀岩姿态），姿态的模式匹配（ai动作生成后自动关联到技术要点），图像处理（站在地面拍高墙，需要对原始结果做矫正，遮挡去除），视频+图文式社区的前后端，app或网页开发

## xhs上一些关于攀岩社区app的讨论
- 独立社区不看好，社区需要海量用户启动，小众需求难以从xhs这种地方引流

feature
- 需要实时统计岩馆人流量

目前没有找到利用ai对线路进行识别分析并推荐动作姿态的app，几乎全部集中在记录与分享的属性上，可替代性很强，所以全部都不温不火。想做出差异化必须从ai的技术难点上攻克。另外有个想法是，初期尽量期望将它做成自用app，而不是社区属性。从打磨这一个核心功能入手，才有可能做出真正有用的东西。
