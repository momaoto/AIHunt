# 蓝图

做这个账号主要是两个目的，一是打通自己的知识链路。我们身处AI行业并且对此抱有热忱，主业是做技术和算法，但对商业应用、落地转化等场景了解较少（且我们很需要这个），最近的AI更新迭代地让人有些恐慌，所以我希望能够主动地迎接这一波浪潮，以内容输出为目的倒逼自己跟上最新的AI前沿；二是根据我对自身优势的认知和对社会讨论的观察，目前有很多像我一样的年轻人想要积极地参与到这一波变革中，但很多人是外行，无从下手，而我对大模型有初步认知，对算法和数据也较为了解，我的合伙人（我最最最可爱的宝贝thk蛋）对新资讯有天生的敏锐度，对海量信息中的有效内容有很强的嗅觉能力，所以我希望我们一个人担任信息的搜集官，一个人担任内容的处理器，共同作为技术和市场的桥梁，为大家打破信息差，打造一个属于我们的，并分享给大家的AI时代知识库。
## 定位
我们的定位是一个寻宝家和机会雷达，不讲技术、模型和算法（或者少讲，用大白话讲），只讲三点
- 最近有什么好用的AI工具
- 某个AI工具怎么用
- 哪些AI赛道有创业机会
本来只想做机会雷达，但是感觉这个需要一些沉淀和积累，我们现在的认知比较短浅，可以先从介绍性内容入手
目前平台上主流AI账号有几种，我们要注意区分自己的独特性
1. 技术讲解类，主要是更新每日paper，讨论算法架构或者一些trick，面向科研人，不做这个，我们要给所有不懂AI的人讲，最好一个专业名词都不要用
2. 资讯分享类，每天更新AI大事，包括模型发布，AI公司的发展，新技术的产生等等，这个偏向于搬运工，我们不是搬运信息，我们搜集信息，然后讲自己的思考和一些话题的提出，比如这个工具暗藏什么商机，做什么已经过时等等
3. 工具讲解类，这个跟我们比较类似，给大家讲什么怎么用，但是还是有点太贴近技术人员了，我们除了讲解，还有机会挖掘和趋势讨论，更贴合产品一些
## 内容形式
内容主要还是想做视频，图文看着费劲，流量不好，视频的话可以做口播或者用AI生成介绍类视频（这个还需要敲定，主要还是吃剪辑功力，我觉得前期做口播简单点）。
每个视频控制在3min左右，每次只讲一个小点，介绍类可以在评论区答疑，讨论类可以放投票等

# 内容选题
## 大方向
### 小众AI工具介绍
这个是我觉得最开始想做的，介绍一些个人开发者的AI工具，打破大家的信息差，这个需要极强的新资讯搜集能力，并进行快速试用，然后进行工具介绍和整理
### Agent 生态
主要讲agent工具的落地场景，站在技术侧的方向剖析架构，然后翻译成大白话，告诉大家这个东西怎么样，有没有可以挖掘的落地场景，这个相对有一些技术，但重心仍然在”商业“
### 开源项目搜集
很多商业化是从开源项目开始的，我们可以找找开源项目然后跟大家讨论有没有商业化场景，这个不做刻意要求，每天逛逛github随缘
### 趋势分析和热点讨论
这个主要是对一些大的话题进行抛砖引玉，比如”Agent 是真趋势还是过渡形态？“，比如”多模态能力会不会压缩很多创业空间？“等等，这个可以等做成熟了再弄
